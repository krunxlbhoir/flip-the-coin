# Flip The Coin

A simple coin flipping game built with HTML, CSS, and JavaScript.

## Demo
https://krunxlbhoir.github.io/flip-the-coin/

## How to run locally
Open `index.html` in a web browser.

## Notes
- Uses JavaScript `Math.random()` for coin flip logic  
- DOM manipulation with `querySelector` and `addEventListener`  
- Responsive layout using Flexbox  
- Button and heading shadows used for depth and visual appeal
